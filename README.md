## Rock, Paper or Scissors

A small game that developed with nodejs called "Rock, Paper or Scissors". There is a perfect artificial intelligence in the game :)

#### Used Libraries
- prompt-list
- mocka
- babel


#### Running
```bash
yarn install
yarn start
```

#### Testing
```bash
yarn test
```
